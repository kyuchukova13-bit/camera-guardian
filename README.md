# camera-guardian
Децата трябва да стоят неподвижни като статуи, за да печелят "звезди", докато Вълшебният Пазител ги наблюдава. Проектът включва аудио предупреждения, визуална обратна връзка и режим "Йога на смеха" за активни почивки. Всичко се обработва локално в браузъра (без да се записва видео), което гарантира 100% поверителност.
